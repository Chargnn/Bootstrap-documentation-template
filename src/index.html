First installation

To get started, you need to do the following:

  1. Make sure you have Node and composer installed on your system. Since the doc and the client uses npm and composer to manage package dependencies. If you don't, installing is quite easy, just visit the Node download page and Composer download page.
  2. Have PHP >= 7.1 installed on your system. Make sure you are using PHP 7.1 or later with the client. Older versions are not and will not be tested.
  3. Run npm install in Client folder to install all npm dependencies.
  4. Run composer install in Client folder to install all composer dependencies.
